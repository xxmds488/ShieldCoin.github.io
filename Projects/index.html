Project Initiate

Start Blockchain

The start of the coin and its first life. The genesis block has beeng generated. A lot of work has been done before this point but this is the public release, so what does our coin have at this moment?

Wallets

SHIELD will need somewhere to send them to if you want to store them. At the moment we're only listed on stocks.exchange and you could store them on there but of course people want to have wallets. We will be releasing a Electrum wallet for Windows and a qt-Linux wallet (prebuild+design for the Linux wallet is on it's way)

Project Enlighten

Also important for a starting coin is having people know what what the coin is about. After project initiate (with a lot of overlap) we'll be trying to explain our project as good as we can.

Project Billfold

Windows

We think that it's important for Windows users to also have a QT wallet. This will probably (and hopefully) be finished way before Q1 even starts because of high demand. The QT wallet is the minimum wallet we'll be creating for Windows users

Linux

If you're using Linux you should not be automatically have to use a qt wallet. We'll be creating a prebuild qt Linux wallet in or before Q1 of 2018. We'll also be creating a Electrum wallet for Linux as we understand that there are a lot of people who like to use Electrum over QT

Possibly more

On high demand and if we think we need something we'll be looking into if not creating, wallets for diffrent platforms. We are already thinking about mac OS wallets but will have to look how things turn out.
