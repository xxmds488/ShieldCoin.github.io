Road map

2017 Q4

Project Initiate

Project Initiate is the first project in the life of SHIELD. In this project we have created the genesis block and released a working wallet for Linux and Windows more information can be found on our Medium here.

Project Enlighten

Project Enlighten is about bringing SHIELD to more people. We'll also explain each of our projects in medium and steemit articles and write the first whitepaper. More information about Project Enlighten can be found here.

2018 Q1

Project Billfold

In Project Billfold we'll be releasing some more desktop wallets to make SHIELD usable for a larger audience. More information, like what wallet we'll be creating can be found here.

Project Globalize

Like Project Billfold, project Globalize is about creating more wallets. Diffrent from project Billfold is what we'll be focusing on. Project Globalize will be bringing wallets to smartphones. More information can be found here.

Project Au Courant

Project Au Courant will be the rebranding for SHIELD. We will have finished some more platforms when this is finished and that will make it easier to get everything on the same level. To find our more about the project click here.

2018 Q2

Project Integrate

Project Integrate will be give SHIELD a use. It'll make it possible to send SHIELD over a few diffrent platforms. More information can be found here.

Project Perdu

Project Perdu will be the project that sets SHIELD apart from other privacy coins. More information can be found here.

2018 Q3

Project Boo

Project Boo will be the intergration for PoS and masternodes. More information will be released soon.

Project Smart

Project Smart is where we are implementing RSK smart contracts. RSK smart contracts are similar to Ethereum smart contracts except made for altcoins and bitcoin. More information will be coming soon.

2018 Q4

Project Sharding

Shariding will be a scalibility improvent which has it's origins in Ethereum, it uses validators to validate shards. And decreases the size of the blockchain someone has to store, this might be hard to implement but it would also be a big improvement. More information about project Sharding will be coming soon.

Project SHIELD

Project SHIELD is our main goal currently for our project, it's where we are going to implement quantum proof transactions. More information about this project will be coming soon.

More about the roadmap can be found here.
